# Collaborative Whiteboard | Description

A JavaScript project for a collaborative whiteboard application, utilizing HTML/CSS for the front-end, Node.js with Express and Socket.IO for the back-end, and the Canvas API for drawing functionality.

## Features

- Smooth drawing and erasing functionality using Canvas API
- Add text
- Real-time collaboration via Socket.IO
- Server-side handling using Node.js and Express

## Setup

1. Clone the repo:
```bash
   git clone https://github.com/yourusername/Collaborative-Whiteboard.git
```

2. Install dependencies:
```bash
    npm install
```

3. Run the project:
```bash
    node server.js
```

## Notes
- node_modules/ is excluded via .gitignore
- Make sure you have Node.js installed
